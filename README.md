# Rock & Pour Website

A luxury static website for "Rock & Pour" at The Bowdon Rooms.

## Features
- **Luxury Aesthetic**: Dark mode with elegant gold accents, tailored for a high-end experience.
- **Dynamic Elements**: Countdown timer to the next event.
- **Easy Maintenance**: Update upcoming bands easily via a configuration array in `script.js`.
- **Responsive Design**: Looks great on desktop, tablet, and mobile.
